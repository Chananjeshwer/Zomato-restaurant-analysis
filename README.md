# Zomato Restaurant Analysis 🍽️

## 📊 Project Overview
This project analyzes Zomato restaurant data using Excel to generate insights on ratings, cost, and customer behavior.

## 🔍 Key Insights
- Top-rated restaurants
- Cost vs rating analysis
- Popular cuisines
- Location-based trends

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards

## 📂 Project Files
- Raw dataset (.xlsx)
- Cleaned dataset
- Dashboard file

## ▶️ How to Use
1. Open the Excel file
2. Explore pivot tables and dashboards
3. Analyze trends and insights
